# Super Calculadora

## Tarea 1 (rama fase1)
Calculadora con html, css y javascript.
Instalar dependencias:
``` sh
npm install
```
Ejecutar con:
```sh
npm run live
```

## Tarea 2 (rama fase2)
Servidor http con express
``` sh
npm install
```
Ejecutar con:
```sh
npm run start
```
